## Short Description
A Python engine for calculating **Strokes Gained** in golf, designed to help players analyze and improve their performance using data-driven insights.

## Long Description
This project implements a **minimal, testable mathematical engine** to calculate Strokes Gained (SG), an advanced golf metric that compares a player’s performance on each shot to the expected performance in the same context.

### Features
- Basic SG calculation with optional penalty.
- Unit tests to ensure accuracy and reliability.
- Minimal REST API using **FastAPI** for easy integration.
- Clean, scalable project structure ready for future enhancements (data analysis, round tracking, visualizations, etc.).

### Technologies
Python, FastAPI, Pytest

### Goal
Build a solid foundation for golf analytics software, showcasing disciplined engineering, unit testing, and best development practices.
